# Citations and Attribution

This document tracks the key scientific references that motivate the methods used in NASE.
Repository prose should remain original and concise; references document ideas, not copied wording.

## Core References

1. Coifman, R. R., & Lafon, S. (2006). *Diffusion maps*. Applied and Computational Harmonic Analysis, 21(1), 5-30.
2. von Luxburg, U. (2007). *A tutorial on spectral clustering*. Statistics and Computing, 17, 395-416.
3. Zelnik-Manor, L., & Perona, P. (2004). *Self-tuning spectral clustering*. Advances in Neural Information Processing Systems 17.

## Notes for This Repository

- Diffusion-operator construction and anisotropic normalization follow standard diffusion-map practice.
- Eigengap cutoff is included as a baseline diagnostic.
- Bandwidth-stability truncation is implemented as a practical robustness heuristic for noisy settings.
